HTML & CSS Quick-Reference Guide

HTML: HyperText Markup Language

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.

  • Many HTML elements require both opening and closing tags, also called container tags.
  • Tags that do not contain closing tags are called empty tags.

HTML PAGE STRUCTURE (syntax)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    Text Here
</body>
</html>

BASIC HTML TAGS

01 | HTML Headings

Headings allow you to divide the content of your page into sections. The HTML Header Elements represent six levels of section headings. <h1> defines the most important heading. <h6> defines the least important heading. 

<h1>Heading level 1</h1>
<h2>Heading level 2</h2>
<h3>Heading level 3</h3>
<h4>Heading level 4</h4>
<h5>Heading level 5</h5>
<h6>Heading level 6</h6>

02 | Paragraph Element

The <p> element allows you to create text paragraphs in your website.

<p>Lorem Ipsum is simply dummy text of the printing and typesetting industry.</p>

03 | Line Break Element

The <br> is a stand alone or void tag. It causes the next element or text to display on a new line.

...text goes here <br>
This starts on a new line

04 | Anchor Element - Adding Links

<a> specifies a hyperlink reference href to a piece of content on your own or another website.

<a href="http://www.contact.com/">Contact us here!</a>

05 | Image Element - Adding/Linking Images

The <img> allows you to insert images into your web pages. The <alt> attribute specifies an alternate text for an image. 

<img src="photo.png" alt="Example Photo">

06 | Div Element

The <div> element configures a structural block area or “division” on a web page with empty space above and below. You can use CSS to style and position the divs by adding class and/or id attributes to your div elements. This is the foundation for CSS-based page layouts.

<div class="description">
    <h1>This is the first heading</h1>
    <p>Description Here</p>
</div>

07 | Span Element

<span> is an inline element that is often used as a container for some text. It is used to style parts of the text.

The span element is similar to div; the difference is that div is a block-level element, whereas span is an inline element.

  • Block-level Elements: Always starts on a new line. Intended for holding stand-alone blocks of content, such as paragraphs of text.
  • Inline Elements: intended to hold smaller pieces of content within a larger block of content, such as a few words or a sentence. 
<h2>Some
    <span style="color:red">Important</span>
Message</h2>

★ | HTML LISTS

  • Unordered Lists <ul> — Displays a bullet, or list marker, before each entry in the list. Each list item starts with the <li> tag.

<ul>
    <li>apple</li> <!-- li: list item -->
    <li>banana</li>
    <li>orange</li>
</ul>

  • Ordered Lists <ol> — Displays a numbering or lettering system to itemize the information contained in the list. Each list item starts with the <li> tag.

<ol>
    <li>first item</li> <!-- li: list item -->
    <li>second item</li>
    <li>third item</li>
</ol>

CSS: Cascading Style Sheets

CSS (Cascading Style Sheets) allows you to define styles for web pages. It targets elements using selectors, which allows you to target specific elements and apply the same style to them.

  • There are three ways you can use to implement CSS into your HTML: internal, external, and inline styles.

CSS STRUCTURE (syntax)

Selector {
    Property 1 : value;
    Property 2 : value;
    Property 3 : value;
}

Applying CSS into HTML

01 | Inline CSS

To apply a distinct style to a single HTML element. Inline CSS makes use of an HTML element's <style> attribute. 

<h1 style="color:blue;">Hello World!</h1>

02 | Internal CSS

To define a distinct style to a single HTML page. Internal CSS is defined in the <head> section of an HTML page, within a <style> element. 

<style>
body { 
    background-color: powderblue;
}
</style>

03 | External CSS

Many HTML pages use an external style sheet to customize their website. To use an external style sheet, include a link to it in the <head> section of each HTML page. 

<head>
    <link rel="stylesheet" href="name of the css file.css">
</head>

CSS Selectors

CSS selectors are used to specify/target the HTML elements to be styled. The three simple types of CSS Selectors are element, class, and id selector.

★ | Element Selector

Used to select HTML elements based on their names. (Example: h1, p, a)

h1 {
    color: blue;
}

★ | ID Selector

The id selector uses an HTML element's id attribute to select a specific element. There can only be one element with the same id.

Syntax: <p id="heading"> Hello World! </p> 

#heading {
    text-align: center;
}

★ | Class Selector

To target only a subset of elements, without changing the other. Can be used multiple times within the page.

Syntax: <h1 class="pop-up"> This is a text </h1> 

.pop-up {
    text-align: center;
}

Styling Text with CSS

Property Uses/Examples
font-family font-family: Arial, "Helvetica Neue", Helvetica, sans-serif;
font-style font-style: italic;
font-weight font-weight: bold;
text-align text-align: justify;
  • Text-Transform property- allows you to transform text with the following values: uppercase, lowercase, capitalize
  • Text-Decoration property- specifies the decoration added to text and contains the following properties: none, underline, overline, line-through
  • Text-Shadow property- allows you to set a shadow for the text.

    Example: h1 { text-shadow: 3px 2px 3px red; }
    • First value- horizontal offset of the shadow.
    • Second value- the vertical offset.
    • Third value- blur radius: a higher value means the shadow is dispersed more widely.
