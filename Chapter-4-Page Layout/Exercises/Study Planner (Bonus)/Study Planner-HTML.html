Teaching Materials

Beginner Level

Introduction to HTML

HTML stands for HyperText Markup Language. It is the most basic building block of the Web. It defines the meaning and structure of web content. HTML consists of a series of elements that tells the browser what and how to display.

Beginner Level

Introduction to CSS

CSS stands for Cascading Style Sheets. It allows you to define styles for web pages. CSS uses selectors to target specific elements and apply the same style to them. Additionally, it uses rules to style elements.

Beginner Level

Introduction to JavaScript

JavaScript is used to add functions to a website. You can add JavaScript code in an HTML document using the <script> tag. The console is part of the web browser and allows you to log messages, run JavaScript code, and see errors and warnings.
